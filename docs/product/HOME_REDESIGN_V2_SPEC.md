# Home Redesign V2

目标：把首页从“展示型 demo”收回到真实工具产品的本地库/启动器。

## 首屏结构

- 顶部只保留本地库标题、总数、游戏/应用数量和继续入口。
- 搜索栏紧跟标题，作为主要入口。
- 分类使用三段筛选：全部、游戏、应用。
- 列表从首屏开始露出，条目密度优先于装饰；不额外放摘要行或分组标题抢空间。
- HdsTab 只做浮动导航，内容需要延伸到其后方，不预留整块底部背景。

## 列表规则

- 每行高度控制在 68-76vp。
- 左侧仅保留稳定的 MRP 类型标识。
- 主信息：名称 + 文件名。
- 次级状态放右侧：类型、最近。
- 游戏/应用区分通过三段筛选、左侧 MR/APP 标识、右侧类型标签共同完成。
- 固定、详情、批量管理不在主列表直接露出，后续放更多操作或详情页。

## 验收

- 首屏至少露出 5 个条目。
- 右侧状态文字不能被裁切。
- 底部浮动栏后面必须能看到列表内容。
- 页面不能出现营销式说明文案。
- 不用大面积渐变卡片堆叠。
