# 一期 MVP 范围

## 一期必须做
- App 基础壳工程
- 本地导入 MRP 包
- 单实例运行
- 基础画面渲染
- 基础输入映射
- 基础音频能力
- 基础存档/配置持久化
- 退出/恢复/异常提示
- 首批代表性样例验证
- 包内资源读取
- gzip 压缩资源解包
- 运行时日志与兼容性定位
- 本地文件合法性提示

## 一期不做
- 大规模兼容库
- 在线内容平台
- 云存档/账号体系
- 多开/悬浮窗
- 高级滤镜与复杂调试器
- 全机型深度适配
- 大规模产品化 UI 精修
- 第三方游戏包分发
- 内置下载中心
- 订阅制商业化

## 一期验收口径
- 可导入
- 可启动
- 可基本交互
- 可退出后再次进入
- 基础存档可用
- 基本日志可定位问题
- 代表性游戏主流程基本可玩
- 不因 MRP 调用退出而杀死宿主 App
- 包内脚本、图片、插件资源能被正确读取

## 商业化边界
- 一期可以免费发布或内部测试。
- 不售卖游戏内容。
- 后续收费优先采用 Pro 一次性解锁。
- 付费点应放在按键布局、皮肤、存档管理、批量导入、每游戏配置、日志导出等工具能力上。
